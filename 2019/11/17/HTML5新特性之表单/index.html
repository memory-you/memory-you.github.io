HTML5表单新特性

发表于 2019-11-17 | 分类于 HTML
本文字数： 2k | 阅读时长 ≈ 2 分钟

HTML5给表单带来了许多新特性，总结为：

  1. 新的input type
  2. 新的表单标签
  3. 表单标签新的属性

新的 Input type

在之前的HTML标准中，Input的type类型有：

值 描述
button 定义可点击按钮。
checkbox 定义复选框。
file 定义输入字段和“浏览”按钮，供文件上传。
hidden 定义隐藏的输入字段。
image 定义图像性式的提交按钮。
password 定义密码字段，该字段中的字符被掩码。
radio 定义单选按钮。
reset 定义重置按钮，重置按钮会清除表单中的所有数据。
submit 定义提交按钮，提交按钮会把表单数据发送到服务器。

而HTML5拥有多个新的表单输入类型，这些新特性提供了更好的输入控制和验证。

Input type description
email 邮件输入域，在表单提交时提供简单的邮箱格式验证，并弹出个窗口。
url URL地址输入域，在表单提交时提供简单的URL地址验证，并弹出个窗口。
number 数字输入域，在表单提交时提供简单的数字格式验证，并弹出个窗口。
tel 电话号码输入域，在手机浏览器中弹出数字输入键盘。
range 范围选择控件，帮助用户在一定范围内选择一个数字。
Data pickers 日期选择控件
search 搜索输入域，在手机浏览器中右下角呈现搜索按键。
color 颜色选择控件，浏览器并未自己实现颜色选择框，而是使用操作系统自带的颜色选择控件。

新的表单标签

在之前的HTML标准中，表单标签有：

Element Description
<input> 根据不同的type属性，可以变化为多张形态。
<textarea> 定义多行输入字段（文本域）。
<select>/<option> 定义下拉列表。
<button> 定义可点击的按钮

HTML5新增表单元素——用于信息提示，不能用于数据提交：

Element Description
<datalist> 数据列表，配合option使用，本身为不可见元素，为普通的input提供输入建议列表
<keygen> 密钥对生成器。当提交表单时，会生成两个键，一个是私钥，一个公钥。
<output> 输出，用于描述表单中的计算结果，语义标签，样式与SPAN无异。
<progress> 进度条，未指定value属性则显示为“进行中”样式；若指定了value（默认在0~1之间）就可以控制其显示的进度
<meter> 刻度尺/度量衡，用红黄绿三色表示出一个数值所处的范围：不可接受/可以接受/最优范围

新的表单属性

Props Description
value 规定输入字段的初始值。
readonly 规定输入字段为只读。
disabled 规定输入字段是禁用的，被禁用的元素不会被提交。
size 规定输入字段的尺寸。
id 定义元素的唯一id。
class 为html元素定义一个或多个类名。
style 规定元素的行内样式。
title 描述了元素的额外信息。
checked 在radio和checkbox中，被选中。
selected 在select中，被选中。

HTML5新增的属性：

Props Description
autocomplete 自动补全，是否自动记录之前提交的数据，以用于下一次输入建议。
placeholder 占位符，用于在输入框中显示提示性文字，与value不同，不能被提交。
autofocus 自动获得输入焦点。
multiple 是否允许多个输入值，若声明了该属性，输入框中就允许输入用逗号分隔的多个值。
form 为一个元素指定form属性，值为某个表单的ID，则此输入域可以放到表单的外部
required 在表单提交时会验证是否有输入，没有输入则弹出提示消息
maxlength 最大长度，在有输入的情况下，限定输入域中字符的个数
minlength 最小长度，在有输入的情况下，限定输入域中字符的个数，不是HTML5标准属性，仅部分浏览器支持
min 限定输入的数字的最小值
max 限定输入的数字的最大值
step 限定输入的数字的步长，与min属性连用
pattern 指定一个正则表达式，对输入进行验证
